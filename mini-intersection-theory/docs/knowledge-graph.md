# Knowledge Graph: Mini Intersection Theory

## L1: Definitions (Complete ✅)
- `Variety` (Type class with dim, isSmooth, isProjective, isComplete, isIntegral, isRegular)
- `Subvariety` (Structure: carrier, embeding, codim, dim_add_codim_eq)
- `Divisor` (Subvariety of codimension 1)
- `RationalFunction` (Zeros and poles as divisors)
- `Morphism` (Proper, flat, relative dimension)
- `AlgebraicCycle` k X (Formal Z-linear combination of k-dimensional subvarieties)
- `ChowGroup` CH_k(X) = Z_k(X) / ~_rat
- `ChernClass` c_i(E) in CH^i(X)
- `VectorBundle` E → X of rank r
- `ProjectiveSpace` P^n, `AffineSpace` A^n
- `Hypersurface`, `CompleteIntersection`
- `LinearSubspace`, `MultiProjectiveSpace`
- `BlowupData`, `NormalConeData`, `DeformationSpace`
- `IsCohenMacaulay`, `IsGorenstein`, `IsLCI`

## L2: Core Concepts (Complete ✅)
- Proper pushforward f_* : Z_k(X) → Z_k(Y)
- Flat pullback f^* : Z_k(Y) → Z_{k+d}(X)
- Rational equivalence: alpha ~ beta via principal divisors
- Proper intersection: codim(Y∩Z) = codim(Y) + codim(Z)
- Intersection multiplicity (Serre's Tor formula)
- Degree of 0-cycles
- Effective cycles (non-negative coefficients)
- Fundamental cycle [V] of a subvariety
- Cycle support, coefficient map
